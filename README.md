# 🎵 Spotify Clone

A responsive Spotify-inspired music streaming homepage built using HTML and CSS. This project was created to practice responsive web design, Flexbox, CSS layouts, and modern UI development.

## 🚀 Features

- Responsive UI
- Spotify-inspired layout
- Sidebar Navigation
- Music Cards
- Hover Effects
- Sticky Music Player
- Clean Code Structure

## 🛠️ Tech Stack

- HTML5
- CSS3
- Flexbox
